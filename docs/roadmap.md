Внеси следующие доработки в проект:

Модуль при выполнении опредлененных условий запускает поиск по внешним директориям и добавляет его результат к результатам основного поиска.
Если результаты дополнительного поиска не вписываются в бюджет, в ответ добавляется перечень директорий, в которых дополнительно можно произвести поиск.

Исключи из этого перечня директорий те, в которых:
- не нешлошсь результатов,
- все результаты поиска уместились в бюджет.
Гарантируй при этом, что дополнительный поиск запускается по всем внешщним директоиям, даже если поиск по предыдущим набрал необходимый лимит (это нужно для корректного исключения директорий, в которых нет )
