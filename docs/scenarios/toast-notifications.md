# Toast-уведомления об ошибках

При инициализации плагин показывает toast-уведомления через OpenCode API (`client.showToast`) для не-фатальных проблем. Порядок проверок:

| # | Условие | variant | Действие |
|---|---|---|---|
| 1 | `directories` пуст или отсутствует и `compile_commands_dir` не указан | warning | Возврат `{}` (плагин неактивен) |
| 2 | Ошибка парсинга opencode.json | error | Продолжение поиска других конфигов |
| 3 | opencode.json не найден | warning | Продолжение работы |
| 4 | Директория из списка не найдена | warning | Одна на каждую отсутствующую |
| 5 | `compile_commands.json` не найден | error | Продолжение работы (только config-директории) |
| 6 | Ошибка парсинга `compile_commands.json` | error | Продолжение работы (только config-директории) |
| 7 | Нет активных директорий (все отсутствуют или disabled) | warning | Возврат `{}` (плагин неактивен) |
| 8 | rg не найден | warning | Grep ограничен, glob работает |
| 9 | zod не найден | warning | deps_read недоступен |
