# Toast-уведомления об ошибках

При инициализации плагин показывает toast-уведомления через OpenCode API (`client.showToast`) для не-фатальных проблем. Порядок проверок:

| # | Условие | variant | Действие |
|---|---|---|---|
| 1 | `directories` пуст или отсутствует, `compile_commands_dir` не указан **и** `.ext-search.json` не найден | warning | Возврат `{}` (плагин неактивен) |
| 2 | Ошибка парсинга opencode.json | error | Продолжение поиска других конфигов |
| 3 | opencode.json не найден | warning | Продолжение работы |
| 4 | Директория из списка не найдена | warning | Одна на каждую отсутствующую |
| 5 | `compile_commands.json` не найден | error | Продолжение работы (только config-директории) |
| 6 | Ошибка парсинга `compile_commands.json` | error | Продолжение работы (только config-директории) |
| 7 | Ошибка чтения `.ext-search.json` | error | Продолжение работы (config- и cc-директории) |
| 8 | Ошибка парсинга `.ext-search.json` (невалидный JSON) | error | Продолжение работы (config- и cc-директории) |
| 9 | Некорректная структура `.ext-search.json` (нет `user_dirs` или не массив) | error | Продолжение работы (config- и cc-директории) |
| 10 | User-директория невалидна (не найдена, не директория, внутри configDir, конфликт с существующей, не строка) | error | Одна на каждую невалидную, продолжение работы |
| 11 | Нет активных директорий (все отсутствуют или disabled) | warning | Возврат `{}` (плагин неактивен) |
| 12 | rg не найден | warning | Grep ограничен, glob работает |
| 13 | zod не найден | warning | deps_read недоступен |
